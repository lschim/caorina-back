# tcm-assistant-back

**Backend API for Chinese Medicine Practitioners**  
A Spring Boot application to assist practitioners in managing herbs, drugs, categories, and other aspects of Traditional
Chinese Medicine (TCM).

---

## ⚡ Quick Start (Development)

1. Clone the repository
2. Use `make build` to build the solution
3. Use `docker compose up -d` to start the dev environment and `make tests` to run the tests (requires the dev
   environment to be up)
